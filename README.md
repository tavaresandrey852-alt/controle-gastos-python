# Controle de Gastos em Python
Sistema desenvolvido para gerenciamento de gastos mensais via terminal.

#  Funcionalidades:
- Adicionar gastos com descrição e valor
- Listar todos os gastos cadastrados
- Calcular o total mensal

#  Aprendizados:
- Uso de listas e dicionários
- Estruturas de repetição (while, for)
- Condicionais (if/elif/else)

# Tecnologias:
- Python
